# Claude WebKit

Skill para crear landing pages profesionales sin programacion por Hainrixz.

## Origen
- GitHub: https://github.com/Hainrixz/claude-webkit
- Instalacion original: `git clone https://github.com/Hainrixz/claude-webkit.git && cd claude-webkit && claude`

## Cuando usar esta skill
- Usuario quiere crear una landing page rapida
- Necesita sitio estatico profesional con HTML/CSS/JS
- Quiere templates de hero, features, testimonials, pricing

## Uso
Desde el directorio de la skill clonada, ejecutar `claude` y solicitar la landing page describiendo el negocio y publico objetivo.
