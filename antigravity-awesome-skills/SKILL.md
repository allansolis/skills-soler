# Antigravity Awesome Skills

Coleccion masiva (1,400+) de skills agenticos curados por la comunidad.

## Origen
- GitHub: https://github.com/sickn33/antigravity-awesome-skills
- Instalacion oficial: `npx antigravity-awesome-skills`

## Cuando usar esta skill
- Usuario busca un skill especifico para un dominio
- Necesita explorar biblioteca de skills por categoria
- Quiere importar skills pre-construidos a su Claude Code

## Uso
Navegar los subdirectorios por dominio y revisar los SKILL.md individuales. Copiar skills a `~/.claude/skills/` segun se necesiten.
