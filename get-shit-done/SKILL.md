# Get Shit Done (GSD)

Framework GSD para proyectos nuevos desde cero.

## Origen
- GitHub: https://github.com/gsd-build/get-shit-done
- Instalacion oficial: `npx get-shit-done-cc@latest`

## Comandos
- `/gsd:new-project` - Iniciar proyecto nuevo
- `/gsd:discuss-phase` - Fase de discusion/brainstorm
- `/gsd:plan-phase` - Fase de planificacion detallada
- `/gsd:execute-phase` - Fase de ejecucion paso a paso
- `/gsd:verify-work` - Verificacion de calidad
- `/gsd:ship` - Publicar resultado final

## Cuando usar esta skill
- Usuario inicia un proyecto nuevo desde cero
- Necesita guia estructurada de discusion, plan y ejecucion
- Quiere contextos limpios por fase para evitar drift

## Flujo
1. Discutir objetivos
2. Planificar implementacion
3. Ejecutar plan en contextos aislados
4. Verificar resultado
5. Shipping
