# Protege Tu App - Security Hardening Skill

Skill de seguridad para aplicaciones web. Configura RLS, CORS y Security Headers siguiendo mejores practicas de produccion.

## Trigger

Activar cuando el usuario mencione:
- seguridad de la app
- RLS / Row Level Security
- CORS
- security headers
- proteger la aplicacion
- hardening

## Uso

Esta skill contiene 3 prompts predefinidos y configuraciones para:

1. **Row-Level Security (RLS)** - Control de acceso a nivel de base de datos en Supabase/PostgreSQL
2. **CORS** - Restriccion de origenes permitidos para APIs
3. **Security Headers** - 6 headers HTTP de proteccion para produccion

Ver archivos de referencia:
- [rls.md](rls.md) - Configuracion de Row-Level Security
- [cors.md](cors.md) - Configuracion de CORS
- [headers.md](headers.md) - Security Headers de produccion
