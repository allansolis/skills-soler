# Agent Skills (Addy Osmani)

Coleccion de 20 playbooks de ingenieria por Addy Osmani.

## Origen
- GitHub: https://github.com/addyosmani/agent-skills
- Instalacion plugin: `/plugin marketplace add addyosmani/agent-skills`

## Playbooks incluidos
- Test-Driven Development (TDD)
- Security Hardening
- Code Review
- Deployment Workflows
- Debugging
- Refactoring
- Performance optimization
- Architecture decisions
- Documentation
- Y mas

## Cuando usar esta skill
- Usuario necesita guia paso a paso para una tarea de ingenieria
- Busca playbook probado para TDD, security, CI/CD, etc.
- Quiere recetas reutilizables para su flujo de trabajo

## Uso
Consultar el playbook correspondiente a la tarea (buscar por nombre en los archivos del repo) y seguir los pasos documentados.
