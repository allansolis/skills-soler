**Estructura sugerida (copiando estrategia Esmeraldas):**

```

   ├─ Auth: ninguna (Meta firma el payload)
   └─ Response: respond immediately

2. Set: Extract sender_id, message_text, page_id
   └─ Para messaging Meta el shape es:
      entry[0].messaging[0].sender.id
      entry[0].messaging[0].message.text

3. IF: page_id == "100123132505557"  (Autos Soler page)
   ├─ True: continuar
   └─ False: stop (no es nuestra page)

4. HTTP Request: POST a CRM_PUBLIC_URL/api/conversations/upsert
   ├─ Headers: X-Webhook-Secret: $WEBHOOK_SECRET
   └─ Body: {channel, sender_id, message_text, business: "autos_soler"}

5. HTTP Request: POST a CRM_PUBLIC_URL/api/bot/dispatch (o directo al bot)
   ├─ URL: http://localhost:5002/webhook (via tunnel)
   ├─ Headers: X-Webhook-Secret
   └─ Body: {mensaje, user_id}

6. Wait: 1-3 seg (para que el bot procese)

7. HTTP Request: POST a Meta Send API
   ├─ URL: https://graph.facebook.com/v25.0/me/messages?access_token=PAGE_TOKEN
   └─ Body: {recipient:{id:sender_id}, message:{text:bot_response}}
```
