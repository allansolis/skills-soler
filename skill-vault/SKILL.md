# Skill Vault

Agente gratuito open-source que cataloga, analiza y recomienda skills de Claude sin requerir instalaciones adicionales.

## Origen
- GitHub: https://github.com/Hainrixz/skill-vault
- Creador: @soyenriquerocha (comunidad tododeia)

## Uso
Desde el directorio clonado, ejecuta `claude` y utiliza:

- `/vault-add` - Agregar una skill al vault tras analisis de seguridad
- `/vault-search` - Buscar skills por categoria o palabra clave
- `/vault-recommend` - Recomendar skills segun contexto actual
- `/vault-discover` - Descubrir nuevas skills de la comunidad
- `/vault-list` - Listar skills organizadas
- `/vault-stats` - Ver estadisticas del vault
- `/vault-remove` - Eliminar una skill del vault

## Categorias
automation, code-quality, design-ui, documentation, marketing, operations, productivity, security, testing, integration.

## Cuando usar esta skill
Cuando el usuario quiera organizar, catalogar, auditar, buscar o recomendar skills instaladas en su Claude Code.
