# Multica

Framework para ejecutar multiples Claudes en paralelo por multica-ai.

## Origen
- GitHub: https://github.com/multica-ai/multica
- Instalacion oficial: `brew install multica-ai/tap/multica && multica setup`

## Cuando usar esta skill
- Usuario necesita trabajar en varios componentes simultaneamente
- Quiere acelerar refactors multi-archivo con paralelismo
- Necesita coordinar multiples Claudes en un proyecto grande

## Comandos
- `multica setup` - Configuracion inicial
- `multica start` - Iniciar equipo de Claudes
- `multica assign <agent> <task>` - Asignar tarea a un agente
- `multica status` - Ver estado del equipo

## Uso
Dividir un proyecto grande en tareas independientes y asignar cada una a un Claude del equipo para ejecucion en paralelo.
