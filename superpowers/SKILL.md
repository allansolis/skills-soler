# Superpowers

Framework open-source de Jesse Vincent (obra/superpowers) que agrega subagentes y flujos de ingenieria senior a Claude Code.

## Origen
- GitHub: https://github.com/obra/superpowers
- Instalacion oficial: `npx superpowers@latest init`

## Caracteristicas
- Subagentes paralelos con contextos independientes
- Workflow de 7 pasos: Brainstorm, Design, Plan, Implement, Test, Review, Complete
- Skills agenticos: brainstorming, writing-plans, executing-plans, test-driven-development, systematic-debugging, using-git-worktrees, dispatching-parallel-agents, subagent-driven-development, requesting-code-review, receiving-code-review, finishing-a-development-branch, verification-before-completion

## Cuando usar esta skill
Cuando el usuario quiera trabajar en proyectos existentes que requieren modificaciones o bug fixes complejos, o cuando necesite subagentes paralelos trabajando en distintas partes del codebase.

## Uso recomendado
- Proyectos con cambios multi-archivo
- Bug fixes que requieren investigacion profunda
- Implementacion de features con TDD y code review integrado
