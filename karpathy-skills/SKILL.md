# Andrej Karpathy Skills

Skills derivadas de los principios de ingenieria de Andrej Karpathy, por Forrest Chang.

## Origen
- GitHub: https://github.com/forrestchang/andrej-karpathy-skills
- Instalacion plugin: `/plugin marketplace add forrestchang/andrej-karpathy-skills`

## Filosofia
Los 4 principios:
1. Pensar antes de codear
2. Entender el problema de raiz
3. Codigo simple y directo
4. Refactorizar sin miedo

## Cuando usar esta skill
- Usuario quiere que Claude piense antes de escribir codigo
- Necesita enfoque reflexivo para problemas complejos
- Busca patrones de ingenieria senior

## Uso
Invocar la skill antes de tareas de codificacion complejas para forzar analisis profundo del problema antes de la implementacion.
